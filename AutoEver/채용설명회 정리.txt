-- 차량용 NV MAP 구분
항법지도(SD Map(standard map))★★★
	용도 : 내비게이션 구현 목적 지도, 도로 단위의 정보, 경로 탐색/안내 기능
	주요 컨텐츠 : 도로 네트워크 ★★
		- 링크(Link) 도로의 형상 표현 -> 링크를 도로 단위로 구축
		- 노드(Node) 교차로 및 도로 연결점 표현
	정확도 : 약 1~5m
	
ADAS 지도 ★★★
	용도 : ADAS 기능 구현 목적의 지도
	주요 컨텐츠 : 도로의 3차원 기하속성 정보
		- 곡률(Curvature) : 도로 평면 형상의 휨 정도
		- 경사도(Slope) : 도로의 수직형상 구배 정보
	정확도 : +-0.5m	

정밀지도(HD Map) -> 팀에서 하는 역할 x

-아래 정보가 뼈대인듯? ★★★
  도로 : 항법지도의 뼈대를 이루는 가장 기초적인 데이터 (경로 탐색 안내에 활용)
	-> Node,Link로 Network 형성 
  검색 : NV를 검색할때 필요한 POI(지번/새주소 DATA)

  배경 : 디스플레이에 표현되는 배경 -> 시각적 효과
  부가DB : NV검색시 요금정보,안전운행,음성,고속도로정보
  도보/자전거 : NV영역 확장을 위해+
   
  -> 위 정보를 구축 하는것이 원도개발팀의 역할

--지도 배포 프로세스
