인성검사 가이드
  채용형 인성검사 : 과장반응,거짓반응이 있음
		해당 반응들을 가려내는 장치가 있음(faking out)
			-> 잘 골라내야함
		faking out이 핵심 
			->어설프게 머리 굴리면 떨어짐

  인성검사에 대한 흔한 오해
		기업이 개발 x -> 외부 기관에서 출제
			-> 인사티은 인성검사의 결과지를 활용만함
			-> 인사팀은 프로세스 모름
		채용형인성검사는 일반 인성검사와 로직이 다름
		인성검사는 정답과 풀이법이 없음
		일관성이 있어야 한다는 오해임
			->너무 일관정이여도 과장,거짓반응이라 판담함
			-> 인간은 일관될수 항상 일관될수 없기 때문
			-> 그래도 어느정도 일관성 있어야함  
		적절한 일관성은 어캐 할 수있냐
			-> 솔직해야함
			-> 결과를 좋게하기위해 생각할수록 인위적 
			-> 무조건 솔직하게해야함
		어떤게 일관성있는 질문인가
			지각한다, 무단횡단 등 -> 뻔한건로 하진 않음
			지각 한번도 없다 -> 거짓응답 측정
			애초에 다 따지면 -> ㅈ됨 그냥 솔직하게 제발
		컨셉잡으면 ㅈ됨
		1~5 정도의 문항에서 보통,극단에 대한 건 그냥 솔직하게
			-> 솔직하게!!!!!!!!!!!!!!솔직하게!!!!!!!
	준비 tip
	  1. 솔직하게 제발(40%가 과장,거짓으로 떨어짐) 
	  2. 컨디션 조절을 잘해야한다 (영향을 많이 받음)
		-> 난 인성이 훌륭해서 합격하지~라는 편안한 마음으로
	  3. 미리 풀어봐서 안정감을 얻고, 과장,거짓이 있는지 없는지만 체크
		-> 인위적으로 멀하는게 아니라 내 성향을 봐서
			조심만 해야지 이정도만
		-> 미리 풀어보면 합격권인지 아닌지 알려줌
		-> 안정,가능,기대 가 나왔다면 멀 하려 하지 마
			-> 그느낌 그대로
		-> 부족,위험은 불합격권은 아님(유동적으로)
			->부족은 합격할수도, 위험은 불합격 높아짐
			-> 부족만 되도 ㄱㅊ. 위험은 변화줘서 응시 추천
	  응답 신뢰도 점수 유의해야 함
		신뢰도 점수를 제일 먼저봄
		비전형성 : 특이한 사람
			-> 대기업일수록 특이한사람 별루
				ex) 움직이는거 싫어, 운동 조아!
		비 일관성,사회적 바람직성
	 모의인성검사에서 반 생산적 직업활동을 제일 유의해야함
		-> 조직, 단체 활동 할 수 있냐?
		-> 높게 나오면 대기업 싫어할지도
			-> 면접에서도 떨어질 수 있음
	직군별 인성검사(측정하는기업 안하는 기업있음)
		-> 보충적 자료, 참고자료

	정답이 있는 문항은 유념해야함(예외적으로 이건 답이 있음)
 	 	-> 건강 상태 (무조건 건강하다로)
		-> 정신 상태 (나는 거짓말한적 없다 -> yes) 정신적으로문제	

  준비가이드
	모의 한번보고 -> 점수 잘나오면 그 느낌 그대로
		       -> 안좋으면 약간 아주 약간 변화

	무조건 솔직하게, 컨디션 좋게
		-> 오직 "나의 인성은 훌륭하다" 라는 생각에만 집중할것
		-> 편안한 마음으로

	-> 능력이 좋다 보다는 인성적으로 좋지않은사람을 걸러낸다

개요 
 112문항군(336문항) 군당 3개문제
  시간 : 60분? 80분인듯?
  종류 : Ipsative (가장 일치하는지 아닌지)
  필기전형 활용 : screen out
	screent out : 정한 기준 미달 탈락
  면접전형 활용 : 유형별 질문

       새롭고 다양한 경험을 선호한다

	~~
	~~
	~~
