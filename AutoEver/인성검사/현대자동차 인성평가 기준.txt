공통 부분


구라칠땐 살짝살짝 해야함 과하면 거짓반응,과장반응에 걸림


A.응답 신뢰도 : 적절해야함
  비전형성 : 일반적 반응 패턴에서 벗어나는 특이 응답
	-> 점수가 높을 수록 특이응답 많음 -> 신뢰도가 떨어짐
	-> 일반적이지 않음
	-> 대기업 보수적, 평범한 사람 좋아함
	-> 50점이 평균 (30~70사이면 평균)

  사회적 바람직성 : 타인에게 좋은 평가를 받기 위한 응답 경향(과장,거짓반응)
	-> 점수 높은 결과 잘받기위해 진실되지않음
		-> 너무 낮으면 좋은평가 안받으려는 마음(30~70사이면 평균)
  무응답 : 다 응답해야함 (안하는 순간 불리)

B.반 생산적 직업행동 : 점수 높은 ㅈ됨 (30~70사이면 평균)
  반 사회성 : 규정,원칙,지시에 대한 무시 및 반항 및 법적,도덕적 불감증
	  -> 무통제 관련 문항 유의할것 -> 해 뜰때까지 마신다 (통제못함)
	  -> 부모님이 나한테 잔소리 못하게,간섭,규제 사라졌으면!(통제못함)
	  -> 무감각(악플보고 표현의 자유롭다 생각함)
	  -> 정치의 부정,편법(우리도 할수 있지), 드러나지않은범법행위 괜찮다
		
  충동성 : 순간적 감정에 따른 충동적 행동, 결과를 생각하지 않는 무분별한 욕구분출
	  -> 머리보다 가슴이 쌘사람
	  -> 무생각 : 우선순위 안고르고 막할때
	  -> 휴대전화요금 많이나온적있다,외상있으면 좋겠다,
	  -> 무자재 : 쉽게흥분, 앞뒤가리지 않아 실수많다

  무책임 : 대책없는 의무 불이행, 책임 회피
	  -> 무의무 : 일을 대충하고 싶다, 약속못지킨다
	  -> 유도피 : 먼곳을 떠나고싶다, 어려움 해결해줄 타인이 있었으면
	  -> 일을 망쳐도 세상은 괜찮아

  -> 조직의 목표에 반하는 행동으로 재산 절취, 재산 파괴, 정보 오남용, 시간과 자원의 유용, 안전 불감 행동, 근태 불량, 업무 성과 부진, 알콜 및 약물 중독, 부적절한언행

C. 기본성격 : 회사의 인재상, 직무 요건(협력,성실) 조직에 잘 적응하도 높은 성과에 적합한 성격이 있음 -> 면접에 활용됨 ㄷㄷㄷㄷㄷ
	기본성격 분석
		1. 정직성:솔직하고,겸손하게 행동하며,규칙에 따라 생활지향태도
		2. 정서성: 정서적 안정상태,타인 감정에 대한 정서적 유대 정도
		3. 외향성:대인교류에 임하는태도나,자신을표현하고자 드러낸정도
		4. 호감성:타인에게 관용적, 주변사람과 원만하게 지내는 태도
		5. 성실성:목표나 과제 완수를 위해 신중하고 열심히 일하는정도
		6. 개방성:자연과예술선호, 독창적이고 새로운분야에 관심정도
	-> 부족한 성격 면접예상 질문에 넣을것
	-> 합불에 결졍되지 않음

D. 현대 자동차 계열사 인재상 : 이건 총점으로 합/불 결정됨
		-> 기대 이상만 하면됨
		-> 부족 부턴 떨어질수도

	1. 도전:실패를 두려워하지 않고, 신념과의지를 가지고 적극적으로 업무추진
		-> 모험관련문항 (강심장이다, 결과잘몰라도 일단해본다)
		-> 확신관련문항 (후회해본적없다, 미래가 낙관적이다)

	2. 창의:새로운 시각에서 문제를 바라보며, 창의적 사고,행동을 실무에 적용 
		-> 새로움추구(하던일을 계속하면 싫증난다, 권위자의심)
		-> 개선하고자(개선하려했다, 실제로 적용한적있다)

	3. 열정:믿음을 바탕으로 회사/고객을 위해 업무를 주도적으로 수행,책임
		-> 책임감(이메일확인하면바로답장한다, 공동목표리마인드한다
			 나라에서 실시하는데 참가해야한다,)
		-> 워커홀릭(가족 or 주위에서 쉬엄쉬엄하라한다, 완벽주의)

	4. 협력:타조직 존중, 시너지를위해 타조직과 방향성공유.타인과 적극협동
		-> 가장 중요하게봄
		-> 조직중심(청렴하다, 손해볼정도로단체생활,)
			야 왤케 손해보면서 살아~ 이말들어본적 있다
		-> 인내(화낸거 본적이없다,

	5. 글로벌 마인드:글로벌상황에 대한 통창력으로 네트워크 전문성개발
		-> 외국어 역량(영어잘쓴다,해외사이트쓴다,해외드라마)
		-> 관계개방성(외국계기업에서 일하고싶다->퇴사아님 걱정 ㄴ)
		-> 지속적인 외국인이있다.
